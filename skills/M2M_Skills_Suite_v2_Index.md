## Architecture Overview

The M2M Skills Suite is a reusable component library embedded in the Sovereign Architect's Claude Code environment. Each skill is a SKILL.md file that fires automatically when its trigger conditions are met. Skills govern production, quality, governance, and distribution — the full lifecycle of M2M output.

Skills Suite v2 adds the Trust/Liability Gate as the release checkpoint for all output. Every skill now closes with the Seven Must-Haves confirmation before any deliverable distributes.

### SKILL 06 — `m2m-trust-liability-gate` *(NEW — v2)*
**Purpose:** Apply the M2M Trust vs. Liability Gate and Seven Must-Haves standard to any output before it distributes. Self-check layer that fires between creation and distribution — ensures authentic work over polished performance.

**Triggers:**

**Core function:** Release gate. Binary question: does this output build Trust or create Liability? Confirms all Seven Must-Haves present before close.

**The Seven Must-Haves:**
1. Trust — earns the room before asking anything
2. Context — calibrated to recipient, room, and channel
3. Liability — nothing ships that creates exposure
4. Taste — sovereign craft standard, no exceptions
5. Intent — why before what, objective declared
6. Intentional Intent — Architect's 20% distinguishes the work
7. Fractional Legitimacy — execution sourced, close owned, output signed

**Stack position:** Exit gate — fires last, before every distribution event.

**Make.com integration:** Maps to HUMAN_REQUIRED routing when gate fails. AUTO_APPROVE only when all seven pass.
